## 使用

```shell
yarn start # 下载并处理excel
yarn downloadExcel # 下载
yarn walkExcel # 处理excel
```

## excel文件命名
文件名为：任务来源_公告日期_公告号.xlsx，多个excel文件名字相同的情况下在最后加”_序号“

如：河北省市场监督管理局_2023-01-19_2023年第3批.xlsx

* 江西省 - CDATA
